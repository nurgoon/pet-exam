# Исправленные ошибки

## Коммит `ace0d34`

Файл: `src/App.vue`

- Проблема: после завершения теста в некоторых окружениях срабатывал runtime error из-за `crypto.randomUUID()`, из-за чего сценарий завершения ломался.
  - Исправление: добавлен безопасный генератор `createId()` с fallback и замена прямых вызовов `crypto.randomUUID()`.

- Проблема: фронт в dev-режиме мог тихо подставлять заглушки вместо данных из БД при любой ошибке API.
  - Исправление: загрузка экзаменов из API сделана приоритетной; fallback на seed теперь только при явном `VITE_USE_SEED_EXAMS=1`.

- Проблема: при частичном падении API (статистика) пользователь не понимал причину.
  - Исправление: добавлено сообщение `dataLoadError` в UI каталога.

## Коммит `809fb85`

Файл: `vite.config.ts`

- Проблема: локальный фронт мог не видеть backend API при запуске dev-сервера.
  - Исправление: настроен proxy `/api` -> `http://127.0.0.1:8000`.
